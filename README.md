# 🎯 Project Overview

## Business Problem

E-commerce businesses generate large volumes of transactional data, making it difficult to monitor sales performance, profitability, and target achievement manually. Management requires a centralized reporting solution to track key performance indicators, identify sales trends, and support data-driven decision-making.

## Objective

The objective of this dashboard is to:

* Monitor overall sales performance.
* Compare actual sales against targets.
* Analyze profit margins across categories and sub-categories.
* Identify top-performing products and regions.
* Track monthly sales trends.
* Support strategic business decisions through data visualization.

## Target Audience

* Executive Leadership
* Sales Managers
* Business Analysts
* Operations Team
* Regional Sales Teams

# 🧠 Data Model & DAX

## Model Type

**Star Schema**

The model follows a star schema design to improve query performance and simplify reporting.

## Fact Tables

### Order Details

Contains transactional sales information including:

* Order ID
* Product Details
* Quantity
* Amount
* Profit

## Dimension Tables

### Orders

Contains:

* Order Date
* Customer Information
* City
* State

### Sales Target

Contains:

* Category Targets
* Segment Targets

### Date Table

Used for time intelligence calculations such as YTD Sales.

---

## Calculated Columns

### Category Type

Combines Category and Sub-Category.

### Revenue Per Order

Calculates revenue generated by each order.

### Sales Category

Classifies orders as:

* Above Average
* Below Average

based on sales amount.

---

## Key Measures

### Total Sales

```DAX
Total Sales =
SUM('Order Details'[Amount])
```

### Total Target

```DAX
Total Target =
SUM('Sales Target'[Target])
```

### Order Count

```DAX
Order Count =
DISTINCTCOUNT('Order Details'[Order ID])
```

### Average Profit Delhi

```DAX
Average Profit Delhi =
CALCULATE(
    AVERAGE('Order Details'[Profit]),
    Orders[City] = "Delhi"
)
```

### YTD Sales

```DAX
YTD Sales =
TOTALYTD(
    [Total Sales],
    'Date Table'[Date]
)
```

### Profit Margin

```DAX
Profit Margin =
DIVIDE(
    SUM('Order Details'[Profit]),
    SUM('Order Details'[Amount])
)
```

# 💡 Key Insights

## Trend A

Electronics generated the highest sales revenue and consistently exceeded target values.

## Trend B

Bookcases, Sarees, and Printers contributed significantly to overall revenue.

## Trend C

Sales peaked during April and October, indicating seasonal demand patterns.

## Trend D

Indore and Mumbai recorded the highest order volumes among all cities.

## Recommendation

Focus marketing and inventory investments on high-performing categories and cities while improving promotional strategies for lower-performing segments.
